# FUTERES MOEX INFO
## Функция получения информации по фьючерсу с Московской биржи
Функция реализует получение информации по фьючерсам, торгуемым на Московской бирже, через запросы на Программный интерфейс
информационно-статистического сервера Московской Биржи. Передаются описание полей и значения, которые размещаются на странице фьючерса на сайте Московской биржи.
В аргументе передается тикет фьючерса формата 'RIZ2', функция возвращает словарь вида:
```
{ '<Код поля>' : {'ru_name': '<Описание на русском>', 'value': '<Значение>'}, '<Код поля>' : {'ru_name': '<Описание на русском>'} }
```
Пример, вывод биржевого сбора:
```
from futures_moex_info import func_futures_moex_info
name, volume = func_futures_moex_info('RIZ2').get('BUYSELLFEE').values()
print(name, volume
```
Вывод:
```
Сбор за регистрацию сделки*, руб. 8.51
```
Пример, вывод всех значений:
```
from futures_moex_info import func_futures_moex_info
data = func_futures_moex_info('RIZ2')
for keys in data:
    print('Код: ' + keys + ' =-= Название: '+ data.get(keys).get('ru_name') + ' =-= Значение: ' +str(data.get(keys).get('value')))
```
